## 🧠 About

**FALUNA: Panic Console** is a fast-paced browser mini-game where you must react quickly to system alerts and keep everything under control.

As Faluna, you are managing a critical console system under pressure.  
Handle alerts, prevent failures, and survive for 60 seconds.

---

## ⚡ Features

- 🧩 Real-time interaction gameplay
- ⏱️ Increasing difficulty over time
- 🚨 Alert / Warning / Error system
- 🎯 Score tracking
- 🎨 Stylish cyber UI

---

## 🎮 How to Play

1. React to alerts as quickly as possible  
2. Prevent system failures  
3. Survive for 60 seconds  
4. Try to beat your best score
